# Back to Work

A 100% Powershell APP with JSON Based Data using WPF as GUI.

## Overview

- 100% Powershell
- Data JSON Based
- Customise the look of the APP XAML
- Markdown support for incident messages
- Multi-language support


## Requirements

- Powershell < V5 lower could work but not tested
